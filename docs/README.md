## 🚀 快速开始

### 第一次使用

1. **⭐ 配置API池（推荐）**
   - 打开 `API池管理工具.html`
   - 添加一个或多个YouTube API Key
   - 所有工具将自动共享这些API

2. **打开工具**
   - 双击任意HTML文件在浏览器中打开
   - 推荐按顺序使用：API管理 → 01 → 02 → 03

3. **四步工作流**
   - **步骤0**: 用`API池管理工具`添加和管理API Key（一次性设置）
   - **步骤1**: 用`01 低粉爆款挖掘`搜索发现优质博主
   - **步骤2**: 用`02 博主管理工具`保存和管理博主
   - **步骤3**: 用`03 视频监控工具`持续追踪博主动态

### 从旧版本升级

1. **自动迁移**
   - 首次打开新版本时会自动迁移数据
   - 无需手动操作

2. **确认迁移**
   - 打开 `../utilities/存储状态检查.html`
   - 查看数据是否迁移成功

3. **清理旧数据（可选）**
   - 使用 `../utilities/快速修复工具.html`
   - 点击"清理 localStorage"

## 📊 存储说明

### IndexedDB vs localStorage

| 特性 | localStorage (旧版) | IndexedDB (新版) |
|------|---------------------|------------------|
| 存储容量 | 5-10 MB | 50 MB - 1 GB+ |
| 性能 | 同步，会阻塞 | 异步，不阻塞 |
| 适用场景 | 简单配置 | 大量数据 |
| 当前使用 | ❌ 已弃用 | ✅ 正在使用 |

### 数据存储位置

- **API Key**: IndexedDB > settings
- **博主数据**: IndexedDB > channels
- **旧数据**: localStorage（可安全删除）

## 🎯 使用技巧

### 01 低粉爆款挖掘工具

#### 高效使用

1. **设置筛选条件**
   - 最大粉丝数：如 100000
   - 最小观看数：如 10000
   - 发布时间：最近一周

2. **选择模式**
   - 点击"进入选择模式"
   - 批量选择感兴趣的视频
   - 获取对应博主链接

3. **导出链接**
   - 选择视频后点击"获取博主链接"
   - 复制链接到博主管理工具

#### 最佳实践

✅ 使用具体的关键词（如"AI绘画"而不是"AI"）
✅ 定期更换搜索关键词
✅ 结合多个筛选条件
✅ 导出有价值的博主链接

### 02 YouTube博主管理工具

#### 高效使用

1. **单个添加**
   - 适合精选博主
   - 立即获取完整信息

2. **批量导入**
   - 准备好频道ID列表
   - 每行一个ID
   - 自动去重

3. **标签管理**

4. **数据维护**
   - 定期点击"刷新"更新数据

   - 定期导出备份

### 03 对标视频监控工具

#### 功能特点

1. **批量监控**
   - 自动读取已保存的博主列表
   - 批量获取每个博主最新视频
   - 实时显示爬取进度

2. **智能排行榜**
   - 观看数排序
   - 发布时间排序
   - 点赞数排序
   - 评论数排序
   - 互动率排序

3. **多维度筛选**
   - 按地区筛选
   - 按博主筛选
   - 按时间范围筛选

4. **可视化展示**
   - 卡片式视频展示
   - 排名标识
   - 爆款标记
   - 互动率分析

#### 使用流程

1. **准备工作**
   - 先在"博主管理工具"中添加要监控的博主
   - 确保API Key已配置

2. **开始监控**
   - 设置每个博主爬取的视频数（默认5个）
   - 点击"🚀 开始监控"
   - 等待自动爬取完成

3. **分析结果**
   - 使用排序方式查看热门视频
   - 使用筛选器定位目标内容
   - 点击视频卡片跳转观看

4. **最佳实践**
   - 定期运行监控，掌握对标博主动态
   - 关注高互动率视频，发现优质内容
   - 结合地区和时间筛选，精准定位

#### 排序方式说明

- **观看数**：找最火的视频
- **发布时间**：看最新的内容
- **点赞数**：找观众最喜欢的
- **评论数**：找讨论度最高的
- **互动率**：找性价比最高的内容（互动/观看）

## 🔐 API Key 说明

### 如何获取

1. 访问 [Google Cloud Console](https://console.developers.google.com/)
2. 创建新项目或选择现有项目
3. 启用 "YouTube Data API v3"
4. 创建凭据 > API密钥
5. 复制密钥到工具中

### API 配额

- **每天配额**: 10,000 单位
- **搜索操作**: 100 单位/次
- **获取频道信息**: 1 单位/次
- **获取视频信息**: 1 单位/次

### 节省配额建议

- 避免频繁搜索
- 批量导入时适当延时
- 使用筛选条件减少请求
- 一个API Key可多个工具共用

## 🛡️ 数据安全

### 数据存储位置

✅ **本地存储** - 数据保存在您的浏览器中
✅ **不上传** - 不会发送到任何服务器
✅ **隐私保护** - 只有您能访问

### 注意事项

⚠️ **不要在隐私模式使用** - 数据会在关闭窗口后删除
⚠️ **不要清除浏览器数据** - 会删除所有存储的数据
⚠️ **更换设备前导出数据** - 数据不会跨设备同步
