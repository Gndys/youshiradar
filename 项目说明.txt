🚀 快速开始：

1️⃣ 配置API
   → 打开 "API池管理工具.html"
   → 添加你的 YouTube API Key
   
2️⃣ 开始使用
   → 打开 "01 低粉爆款挖掘 .html" - 发现爆款视频
   → 打开 "02 YouTube博主管理工具.html" - 管理博主
   → 打开 "对标监控/chuhai.html" - 监控最新视频
