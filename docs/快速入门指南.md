## 🚀 5分钟上手流程

### 第1步：准备API Key（2分钟）

1. 访问 [Google Cloud Console](https://console.developers.google.com/)
2. 创建项目 → 启用"YouTube Data API v3"
3. 创建凭据 → API密钥
4. 复制密钥备用

**提示：** 如果觉得麻烦，工具内置了5个API密钥池，开箱即用！

---

### 第2步：发现优质博主（1分钟）

**使用工具：** `01 低粉爆款挖掘.html`

```
1. 打开工具
2. 输入搜索关键词（如"AI绘画"）
3. 设置筛选条件：
   - 最大粉丝数：100000
   - 最小观看数：10000
   - 发布时间：最近一周
4. 点击"开始分析"
5. 进入"选择模式"，勾选感兴趣的视频
6. 点击"获取博主链接"
```

**输出：** 获得一批优质博主链接

---

### 第3步：建立博主库（1分钟）

**使用工具：** `02 YouTube博主管理工具.html`

```
1. 打开工具
2. 点击"批量导入"
3. 粘贴刚才复制的博主链接
4. 点击"开始导入"
5. 等待自动获取博主信息
6. （可选）添加标签分类
```

**输出：** 建立了自己的博主库

---

### 第4步：监控最新动态（1分钟）

**使用工具：** `03 对标视频监控工具.html`

```
1. 打开工具（会自动读取博主库）
2. 设置每个博主爬取5个视频
3. 点击"开始监控"
4. 等待自动爬取完成
5. 选择排序方式（如"观看数"）
6. 查看排行榜，点击视频观看
```

**输出：** 看到所有博主的最新热门视频

## 🛡️ 注意事项

### ✅ 推荐做法

- 定期备份博主数据（导出CSV）
- 使用标签分类管理博主
- 根据API配额合理安排监控频率
- 关注互动率高的视频，质量更好

### ❌ 避免做法

- 不要在隐私模式下使用（数据会丢失）
- 不要清除浏览器数据（会删除所有存储）
- 不要一次性监控过多博主（API限制）
- 不要频繁重复爬取相同博主（浪费配额）
