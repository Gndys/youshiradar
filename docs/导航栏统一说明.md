## 🔧 技术实现

### CSS架构
```css
.top-nav                    /* 导航容器 */
  .nav-container           /* 内容容器 */
    .logo                  /* Logo区域 */
    .nav-menu              /* 导航菜单 */
      ul > li > a          /* 菜单项 */
    .mobile-menu-btn       /* 移动端按钮 */
```

### JavaScript逻辑
- 使用 `classList.toggle('show')` 切换菜单显示
- 事件委托处理菜单项点击
- 立即执行函数避免全局污染
